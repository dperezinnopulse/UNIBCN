# 07. Perfil de Usuario (Gestor)

## URL
- `perfil.html`

## Funcionalidad
- Ver/editar usuario y email
- Cambiar contraseña (opcional)

## Pasos
1. Abrir `perfil.html`
2. Editar campos necesarios
3. Pulsar “Guardar”

## Selectores Playwright
- Usuario: `#username`
- Email: `#email`
- Nueva contraseña: `#password`
- Guardar: `#btnGuardar`
